# 価格改定見積書・作成ツール 利用マニュアル

本ツールは、得意先別の受注データを読み込み、価格改定後のシミュレーションを簡単に行い、見積書（Excel）を自動生成するためのWebアプリケーションです。

---

## 1. 概要
- **Web上で完結**: インストール不要。ブラウザでURLを開くだけで利用可能です。
- **安心のセキュリティ**: 読み込んだデータは**ブラウザ内でのみ処理**されます。サーバーに送信・保存されることはありません。
- **柔軟なシミュレーション**: 一括での値上げ設定に加え、グループごと、商品ごとの微調整が可能です。

## 2. 操作の4ステップ

### ステップ1：入力用Excelの準備
読み込ませるExcelファイルには、以下の条件が必要です。

- **シート名**: `受注データ` という名前のシートが必要です。
- **必須の列目（ヘッダー）**:
    `受注№`, `種別`, `重量`, `商品コード`, `商品名`, `タイトル`, `形状`, `受注数`, `単価`, `印刷代`, `営G`, `印刷営G`, `材質名称`, `印刷コード`, `表色数`, `裏色数`, `総色数`, `JANコード`, `直送先コード`, `直送先名称`, `最新受注日`, `デザイン名`

> [!NOTE]
> お手元のデータにこれらの列が含まれていることをご確認ください。

### ステップ2：ファイルの読み込み
1. 本ツールのトップ画面にあるアップロードエリアにExcelファイルをドラッグ＆ドロップするか、クリックして選択します。
2. 読み込みが完了すると、画面にデータ件数と概要（売上合計など）が表示されます。

### ステップ3：価格改定のシミュレーション
画面上部のコントロールパネルで条件を設定します。

- **値上げベース**: 「一律（%）」または「一律（円）」を選択し、数値を入力します。
- **端数丸め**: 「なし」または「.00 / .50（0.5円単位）」を選択できます。
- **実施時期 / 前回値上げ日**: 見積書に記載する日付や、古い在庫（前回値上げから時間が経過しているもの）を赤くハイライトするための基準日を設定します。

### 4. 見積書の作成と出力
シミュレーション結果をExcel形式の見積書として出力します。

- **実施時期の入力**: 
    - カレンダーから日付を選択します。
    - **受注分／出荷分**: 日付の横のボタンで、その日から「受注した分」に適用するのか「出荷する分」に適用するのかを選択できます。これらはExcelのフッター文言に反映されます。
- **見積書ダウンロード**: 画面右下の「Excel出力」ボタンをクリックすると、現在の表示内容に基づいたExcelファイルがダウンロードされます。
- **履歴の記録**: 出力した内容は画面下部の「保存済み履歴」に自動的に記録されます。

## 3. 便利な詳細機能

### カテゴリ別タブ
- **別注・ポリ別注 / SP・シルク / 既製・その他 / シール**:
  種別ごとにタブが分かれています。タブを切り替えてそれぞれの進捗を確認できます。

### グループ単価設定（別注・SP）
- 「材質・重量・色数」が一致する商品を自動でグループ化します。
- メニューを展開し、グループごとに一括で単価や営G（営業利益）を設定できます。

### 設定の保存と読み込み
- 作業途中のシミュレーション条件や微調整した値は、画面右上の **「💾 保存」** ボタンで保存（JSON形式）できます。
- 次回作業時に **「📂 読込」** からそのファイルを読み込むことで、前回の続きから作業を再開できます。

### マスタデータの登録（マスター管理）
- カテゴリごとに「標準の単価表」を読み込ませておくことで、受注データに単価がない場合でも自動照合が可能です。
- 画面右上の歯車アイコン（または設定エリア）からマスタファイルを読み込めます。

## 4. よくある質問 (FAQ)

**Q. データが正しく表示されません。**
A. Excelのシート名が `受注データ` になっているか、また必須の列名が正しく入力されているか確認してください。

**Q. どのブラウザが推奨ですか？**
A. Google Chrome または Microsoft Edge の最新版を推奨します。

**Q. 社外秘のデータを入れても大丈夫ですか？**
A. はい。本ツールは「クライアントサイド（ブラウザ）」で動作するため、弊社や外部のサーバーにデータが送信されることは一切ありません。安心してお使いください。
