# 価格改定見積書・作成ツール (Price Quotation Tool)

得意先別のExcelデータを読み込み、別注商品やSP商品の価格改定をシミュレーションし、見積書Excelを自動生成するツールです。

## 🚀 利用方法（Web版）

本ツールはWebブラウザから直接ご利用いただけます。

**[Web版ツールを開く]**

詳細な使い方は **[利用マニュアル (MANUAL.md)](./MANUAL.md)** をご覧ください。

## ✨ 主な機能
- **自動計算**: 全商品の単価を％または一律金額で値上げシミュレーション。
- **別注グループ管理**: 材質・重量・色数が同じ商品を自動でグループ化し、一括で単価設定。
- **営Gインパクト算出**: 単価の変化に伴う営業利益（営G）の変化を自動計算。
- **インライン編集**: 各行ごとに改定単価や営Gを個別に微調整。
- **Excel出力**: そのまま得意先に提出可能な形式の見積書を自動生成。
- **セキュリティ**: データはブラウザ内でのみ処理されるため、サーバーにアップロードされず安全です。

## 🛠 開発者向けセットアップ

ローカル環境や独自のサーバーで実行する場合の手順です。

### 動作環境
- Node.js (v18以上推奨)

### 手順
1.  **依存パッケージのインストール**:
    ```bash
    npm install
    ```
2.  **起動**:
    `起動.bat` をダブルクリック、または以下のコマンドを実行してください。
    ```bash
    npm run dev
    ```
3.  **ブラウザで開く**: [http://localhost:3000](http://localhost:3000) にアクセス。

### テスト・品質管理
- **テスト**: `npm test` でロジックの検証が可能です。
- **CI/CD**: GitHub Actions により、プッシュ時に自動でLintとテストが実行されます。
