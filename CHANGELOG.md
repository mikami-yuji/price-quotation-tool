# CHANGELOG

すべての主要な変更はこのファイルに記録されます。

## [Unreleased] - 2026-04-10 (Update)

### Added
- **専用数値入力コンポーネント**: 入力中の値をローカルで制御し、確定時のみ状態を更新する `InlineNumericInput` を新規作成。

### Changed
- **数値入力の操作性改善**: 数値入力セルのクリック時に全選択する機能を追加。また、入力途中に値が勝手に書き換わったりデフォルトに戻ったりする現象を解消。
- **入力仕様の最適化**: ユーザーの要望に基づき、空欄確定時は `0` を代入、入力中はカンマなしで数値編集ができるように調整。

## [Unreleased] - 2026-04-09 (Update)

### Added
- **ライト/ダークモード切り替え機能**: ビジネス・ブルー基調のクリーンなライトモードをデフォルトとして実装し、ダークモードとの切り替えトグルを追加。
- **テーマ設定の永続化**: ユーザーが選択したテーマをブラウザ（localStorage）に保存する機能を実装。
- **顧客名の略称展開**: Excel出力時、顧客名に含まれる略称（(株)や(㈱)など）を自動的に「株式会社」へ展開する機能を実装。
- **Excel出力の改善（詳細情報の保持）**: 見積書のExcel出力に「営G」および「改定後 営G」列を追加。通常は非表示（折りたたまれた状態）にし、必要に応じて展開して確認できる構成に改善。
- **実施時期の選択機能**: UI上にカレンダー（日付選択）を追加し、Excelの見積書に具体的な実施予定日を反映できる機能を実装。未選択時は「別途ご相談」と表示。
- **直送先情報の表示**: 画面上の一覧およびExcel出力に「直送先コード」と「直送先名称」の列を追加し、配送情報の確認を容易にしました。また、Excel出力時は初期状態でこれらの列を折りたたむ（アウトライン）設定を追加しました。
- **営G改定率の追加**: Excel出力に「営G改定率」列を新設。計算式を埋め込み、営Gの金額変更に連動して率が自動計算されるようにしました。また、他の営G列と同様に折りたたみ設定を適用しました。

### Changed
- **ビジュアルデザインの刷新**: 信頼感のあるビジネスカラー（ブルー/スレート）を基調とした、より専門的なUIデザインへ移行。
- **「目に優しい」配色への調整**: 眩しさを抑えるため、ライトモードの背景を柔らかなオフホワイト（Stone 50系）に変更し、全体のコントラストを長時間作業向けに最適化。
- **視認性の向上**: ライトモード時に文字が背景と同化していた箇所（サマリー数値、入力項目等）の配色を修正し、全てのモードでクリアな表示を実現。
- **数値入力の挙動改善**: number型の入力フィールドにおいて、矢印キー（上下）による数値の増減を無効化。表内のナビゲーション機能を維持しつつ、意図しない数値変更を防止。
- **UIクリーンアップ**: 不要になったフローティングアクションボタン（出力ボタン）を削除し、画面レイアウトを整理。
- **ツール名の正式変更**: 利便性と分かりやすさを考慮し、ツール名を「値上げ見積書・作成ツール」から「価格改定見積書・作成ツール」に変更。

### Added
- **データの並び替え機能**: リスト全体を「材質名称 ＞ 重量 ＞ 色数」の優先順位で昇順に並べ替えるロジックを実装。

### Changed
- **別注・ポリ別注・既製品のエクセル出力最適化**: 「別注」「ポリ別注」に加え「既製品」の場合も、エクセル出力から「印刷コード」と「印刷代」を除外するように変更。
- **エクセル出力の挨拶文調整**: 挨拶文から「拝啓」および「敬具」を削除。長い文章が途切れないようセルの自動折り返し設定を追加するとともに、営G列の折りたたみ時を考慮して行の高さを十分なサイズ（30〜45px）に固定設定。
- **エクセル出力のページ設定改善**: A4横・1枚に収まる（Fit to Page）設定を追加し、印刷時の利便性を向上。
- **エクセル書式設定の堅牢化**: 列の動的な変更に対応するため、列キーに基づいた数値フォーマット適用ロジックに改善。

## [Unreleased] - 2026-04-08 (Update)

### Added
- **形状・受注数・印刷コード列の追加**: UIテーブルおよび見積書PDFに「形状」「受注数」「印刷コード（SPのみ）」列を追加。
- **SP・シルクタブの表示改善**: 商品名の表示ソースをExcelの「タイトル」列（U列）に切り替え可能に。
- **商品名自動短縮機能**: 長い詳細名称（SPタイトル）から記号や記述スペックを自動除去し、核心的なブランド名のみを抽出して表示するロジックを実装。

### Changed
- **UIレイアウトの最適化**: 「印刷代」列を「印刷営G」の直前に移動し、印刷関連の情報をグループ化。
- **PDFレイアウトの調整**: 列の増加に伴い、見積書PDFの列幅を再計算して視念性を確保。

### Fixed
- **ESLintの警告解消**: コード品質向上のため、全ての `any` 型を具体的な型定義へ差し替え、未使用インポートを削除。
- **CIパスの確保**: GitHub ActionsのCIを確実に通過させるためのコードクリーンアップ。
