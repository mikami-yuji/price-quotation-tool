# CHANGELOG

すべての主要な変更はこのファイルに記録されます。

## [Unreleased] - 2026-04-08

### Added
- **別注グループ単価管理機能**: 材質・重量・色数の組み合わせを自動グループ化し、一括で単価設定できるUIを実装。
- **重量別の階層表示**: グループ設定画面にて「5 ㎏」「10 ㎏」などの重量別に見出しを表示し、単位を付与。
- **改定後の「営G」計算機能**: 「旧営G + 単価の値上げ差額」を自動算出し、シミュレーション結果に表示。
- **リスト内インライン編集機能**: 集計テーブル上で「改定単価」と「改定後営G」を直接書き換え可能に。
- **個別修正の強調表示**: リストで直接入力した箇所をオレンジ色の枠線（Amberカラー）でハイライト。
- **折りたたみ機能**: 別注グループ単価設定パネルを開閉可能にし、作業スペースを最適化。
- **PDF出力の連動**: 現在表示中のタブのみを出荷対象とし、社内管理用の列（営G、印刷代等）をPDFから自動で除去。
- **GitHub Actions (CI/CD)**: プッシュ時に自動でテストと静的解析を実行するパイプラインを構築。

### Fixed
- **計算ロジックの優先度整理**: [個別手入力] > [グループ設定] > [デフォルトシミュレーション] の順で適用されるよう修正。
- **表示項目の細分化**: 印刷関連の列を「SP・シルク」タブ限定にし、その他のタブでは非表示になるよう調整。
- **数値の端数処理**: 小数点第2位までの丸め、および要望に応じた 0.5 単位の丸めオプションの精度を向上。
