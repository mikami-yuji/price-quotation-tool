# CHANGELOG

すべての主要な変更はこのファイルに記録されます。

## [Unreleased] - 2026-04-08 (Update)

### Added
- **形状・受注数・印刷コード列の追加**: UIテーブルおよび見積書PDFに「形状」「受注数」「印刷コード（SPのみ）」列を追加。
- **SP・シルクタブの表示改善**: 商品名の表示ソースをExcelの「タイトル」列（U列）に切り替え可能に。
- **商品名自動短縮機能**: 長い詳細名称（SPタイトル）から記号や記述スペックを自動除去し、核心的なブランド名のみを抽出して表示するロジックを実装。

### Changed
- **UIレイアウトの最適化**: 「印刷代」列を「印刷営G」の直前に移動し、印刷関連の情報をグループ化。
- **PDFレイアウトの調整**: 列の増加に伴い、見積書PDFの列幅を再計算して視念性を確保。

### Fixed
- **ESLintの警告解消**: コード品質向上のため、全ての `any` 型を具体的な型定義へ差し替え、未使用インポートを削除。
- **CIパスの確保**: GitHub ActionsのCIを確実に通過させるためのコードクリーンアップ。
